## Getting started
1. Clone the repo
```sh
 git clone
```
2. Build docker image
```sh
 docker build -t portfolio_app .
 ```
3. Run docker image
```sh
 docker run -p 8501:8501 portfolio_app
```
4. Browse to http://localhost:8501 to use the app
